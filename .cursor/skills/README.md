# PlateMyDay Skills

This directory contains custom skills for Cursor agents working on PlateMyDay.

## Available Skills

### update-cursor-md.md
Automatically updates the `cursor.md` file with crucial project information.

**Usage**: Simply say "update cursor.md" or "update the cursor file"

**What it does**:
- Maintains high-impact signals only
- Filters out nitty-gritty details
- Keeps the file under 100 lines
- Focuses on information that saves exploration time

**When to use**:
- After major architectural changes
- When adding critical new patterns or rules
- After significant feature additions
- When new dependencies are added

## Creating New Skills

To create a new skill, add a `.md` file to this directory following the Cursor skills format.
